## Overview
This project helps visually impaired individuals by recognizing common objects and text from real-world scenes using deep learning models.

## Features
- Object detection using TensorFlow and Keras
- Text recognition from images
- Real-time camera-based detection
- Audio feedback for identified items (optional enhancement)

## Tools & Technologies
- **Language:** Python  
- **Libraries:** TensorFlow, Keras, OpenCV, NumPy  
- **IDE:** PyCharm
